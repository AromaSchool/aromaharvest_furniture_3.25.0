為什麼你需要檸檬茶樹（檸檬細籽）精油？
  • # 檸檬茶樹精油所含的香茅醛（Citronellal）具有驅蟲特性，無論是驅趕或者對抗蚊蟲，都有不錯效果。
  • # 淡淡的柑橘香氣，幫助舒緩情緒壓力賦予正面能量，也能調理因壓力引起的消化不良。
哪些人最需要檸檬茶樹（檸檬細籽）精油？
  • # 每到夏天總是變成蚊子目標的人：檸檬茶樹當中的香茅醛有著驅逐蚊蟲的特性，搭配擴香或清潔使用，大方穿上短褲、短裙也不用怕被叮。
  • # 不小心就會太焦慮的人：檸檬茶樹的柑橘清香能使焦慮、緊張情緒冷靜下來，並帶來積極樂觀的正面能量。
使用方式
調油
  • # 成人：精油與基底油調成 1%-1.5% 濃度使用於臉部肌膚，3% 濃度使用於脖子以下身體
  • # 兒童：兩歲以下孩童避免使用精油，即使已經過調油稀釋後也不建議使用
    - 3 歲至 10 歲孩童：精油與基底油調成 1%-1.5% 濃度使用
    - 12 歲以後：精油與基底油調成 2-3% 濃度使用
沐浴
  • # 泡澡：將精油滴 5-8 滴在 5ml 的全脂牛奶或一顆奶球裡混和均勻後倒入浴缸即可
  • # 淋浴：將精油與基底油調和好的按摩油塗抹在身上後即可進行沖澡
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
乳液
  • # 基底油與蘆薈膠以 1:9 的比例調成乳液。作法：基底油倒入容器再加入精油和蘆薈膠後，以攪拌棒打勻後即可
這樣用最好用
情緒調理
  • # 檸檬茶樹 + 純正薰衣草 + 甜橙擴香，幫助緊繃的情緒感到輕鬆，也有助消除憤怒
居家清潔
  • # 檸檬茶樹 + 檜木/扁柏 + 天竺葵室內擴香，用於防蚊
注意事項
  1. 精油不可食用
  2. 未經稀釋不可直接大量使用於皮膚上
  3. 不可直接滴入水中泡澡
  4. 請存放於室內，勿照射太陽、勿存放於冰箱或浴室
  5. 使用時請遠離火源
  6. 嬰兒不建議使用精油
  7. 孕婦若要使用請諮詢醫生建議
  8. 保存期限依瓶身標示，請於期限內使用完畢
