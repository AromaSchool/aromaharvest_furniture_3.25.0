置身貴族氛圍中的優雅香氣
－橙花純露

相傳16世紀，義大利蕾蘿莉（Neroli）郡的女王，非常喜歡橙花的氣味，用它來香薰全身進而受到貴族喜愛，後來人們用女王的來命名了這個植物。 雪白如玉的橙花散發高雅迷人的香氣，其純露氣味與橙花精油幾乎是一樣的華麗和優美，對收斂油脂分泌過度的油性肌膚效果很好。

為什麼你需要橙花純露？
  • # 平衡油脂分泌、活化肌膚
  • # 舒緩情緒焦躁、過動
  • # 舒緩憂鬱情緒、安撫驚嚇感
哪些人最需要橙花純露？
  • # 正處於青春期的人：青春花漾的煩惱除了愛情還有油脂分泌過旺的皮膚，輕輕揮灑橙花純露在臉上，告別滿臉油光青春時光。
  • # 想要優雅轉身的人：不同於玫瑰，橙花純露的香氣更為雅緻，輕輕拍在肌膚上，讓妳轉身時不經意散發出淡淡天然花香。
  • # 經常容易受到驚嚇的人：有時真的不是驚喜，突如其來的消息需要一點時間消化，讓橙花純露壓一壓你焦躁的情緒。
使用方式
護膚
  • # 將純露滴在化妝棉或面膜紙濕敷3分鐘，或噴霧在臉上早晚當化妝水使用。
泡澡
  • # baby澡盆倒入約20ml純露，一般浴缸約50-80ml。
擴香
  • # 滴入 10-20 滴左右的 100% 精油到擴香儀中即可（特別針對呼吸道和情緒）
塗抹
  • # 將純露和蘆薈膠混和均勻即可，製成凝膠比例約3:7，精華液比例約8:2。
這樣用最好用
身體保養
  • # 腹部放鬆：噴灑或濕敷在腹部，可以緩解因壓力造成的消化不良，也可舒緩女性生理期不適
情緒調理
  • # 舒緩情緒：噴灑在身旁周圍可以緩和緊張、壓力和歇斯底里的情緒
肌膚護理
  • # 油性膚質適用
    *建議和玫瑰純露、義大利永久花純露輪替或調和使用。

常見問題
Q1：純露可以喝嗎？
A：沐禾有機純露系列皆通過Ecocert有機認證，並未額外添加防腐劑與抗菌劑，建議未經專業芳療師的指示請勿自行服用。
Q2：純露一定要放冰箱保存嗎？
A：純露是蒸餾精油後所得到的有機水溶液，若溫度變化過大或保存不當可能導致細菌滋生或變質，一般建議冷藏保存以維持品質穩定。
