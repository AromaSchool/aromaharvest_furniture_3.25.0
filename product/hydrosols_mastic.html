－熏陸香純露

熏陸香純露由蒸餾乳香黃連木樹脂所得，主要產自希臘的克里特島，古希臘人曾咀嚼它治療胃痛。不同於乳香，熏陸香屬於漆樹科。適合用來製成口香糖、健胃糖、牙膏、香皂等。

為什麼你需要熏陸香純露？
  • # 收縮血管，緩解靜脈曲張
  • # 有效舒緩胃痛和胃潰瘍
  • # 改善蚊蟲叮咬的痕癢
哪些人最需要熏陸香純露？
  • # 血液循環差的人：熏陸香純露能幫助改善淋巴和靜脈的血液循環
  • # 腸胃敏感的人：熏陸香純露能對抗幽門螺旋桿菌，能緩解胃潰瘍、十二指腸
使用方式
護膚
  • # 將純露滴在化妝棉或面膜紙濕敷3分鐘，或噴霧在臉上早晚當化妝水使用。
泡澡
  • # baby澡盆倒入約20ml純露，一般浴缸約50-80ml。
塗抹
  • # 將純露和蘆薈膠混和均勻即可，製成凝膠比例約3:7，精華液比例約8:2。
這樣用最好用
身體保養
  • # 對於易水腫部位，可以濕敷10分鐘在易腫脹部位，減少不適。
注意事項
  1. 請置於冰箱冷藏保存
  2. 保存期限依瓶身標示，請於期限內使用完畢
  3. 若出現懸浮物質請停止使用
  4. 若需分裝請注意器材務必確實消毒以避免變質
常見問題
Q1：純露可以喝嗎？
A：沐禾有機純露系列皆通過Ecocert有機認證，並未額外添加防腐劑與抗菌劑，建議未經專業芳療師的指示請勿自行服用。
Q2：純露一定要放冰箱保存嗎？
A：純露是蒸餾精油後所得到的有機水溶液，若溫度變化過大或保存不當可能導致細菌滋生或變質，一般建議冷藏保存以維持品質穩定。
