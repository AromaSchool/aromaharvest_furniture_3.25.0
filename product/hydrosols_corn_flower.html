用途多元的淨化高手
－矢車菊純露

矢車菊純露的用途非常多元廣泛，它具備優異的的保濕效果。用於皮膚，能改善乾燥、給予肌膚水樣般的光澤；用於頭髮，可幫助滋潤，使頭髮變得光潤有彈性。
除此之外，它最有名的用途便是可用來保養眼睛，如果眼睛有疲勞不適的情況，都可以用矢車菊來潔淨、緩解。

為什麼你需要矢車菊純露？
  • # 有顯著的保濕效果，賦予肌膚和秀髮亮度及光澤
  • # 放鬆眼周，讓眼睛水潤明亮
哪些人最需要矢車菊純露？
  • # 戴眼鏡或長時間盯著電腦的人：因為用眼過度使得眼睛容易乾澀、疲勞，可以使用矢車菊濕敷眼睛或濕敷皮膚表面。
  • # 因肌膚乾燥而產生細紋的人：可以單獨擦拭或濕敷，也可以加在面膜、面霜及乳液中，具有減少細紋、緊實肌膚的效果。
  • # 頭髮枯黃的人：加在洗髮精或潤絲中，有助於改善髮質。
使用方式
護膚
  • # 將純露滴在化妝棉或面膜紙濕敷3分鐘，或噴霧在臉上早晚當化妝水使用。
泡澡
  • # baby澡盆倒入約20ml純露，一般浴缸約50-80ml。
塗抹
  • # 將純露和蘆薈膠混和均勻即可，製成凝膠比例約3:7，精華液比例約8:2。
這樣用最好用
肌膚護理
  • # 矢車菊純露 + 岩玫瑰純露，可濕敷於眼部周圍，舒緩雙眼疲勞
注意事項
  1. 請置於冰箱冷藏保存
  2. 保存期限依瓶身標示，請於期限內使用完畢
  3. 若出現懸浮物質請停止使用
  4. 若需分裝請注意器材務必確實消毒以避免變質
常見問題
Q1：純露可以喝嗎？
A：沐禾有機純露系列皆通過Ecocert有機認證，並未額外添加防腐劑與抗菌劑，建議未經專業芳療師的指示請勿自行服用。
Q2：純露一定要放冰箱保存嗎？
A：純露是蒸餾精油後所得到的有機水溶液，若溫度變化過大或保存不當可能導致細菌滋生或變質，一般建議冷藏保存以維持品質穩定。
