－杜松純露

杜松純露能提振循環系統，助益腎功能，適合用來處理風濕、關節炎，以及水腫等任何與水分滯留有關的問題。 它的收斂效果也極佳，用於護膚可以幫助清潔及收斂毛孔，很適合油性肌膚使用。

為什麼你需要杜松純露？
  • # 調理痛風、關節炎
  • # 利尿、消水腫、促循環，對抗橘皮組織
  • # 改善毛孔粗大、臉部易出油的問題
  • # 具有潔淨及解毒的特性，能幫助淨化身心及磁場
哪些人最需要杜松純露？
  • # 希望自己身輕如燕：無論是減重還是要讓身體排毒，杜松純露都是極佳的選擇，它能幫助排除身體多餘體液及老舊廢物。
  • # 希望自己的肌膚清爽細膩：妝化再好出門沒多久就會出油脫妝，備有紙巾或吸油面紙是常態，讓杜松純露來幫助你收斂毛孔、平衡皮脂腺，解決油性皮膚的痛！
  • # 希望自己正能量滿滿：幫助驅散心中的烏雲及外界混濁的氣場，杜松可以給予你潔淨清明的能量。
使用方式
護膚
  • # 將純露滴在化妝棉或面膜紙濕敷3分鐘，或噴霧在臉上早晚當化妝水使用。
泡澡
  • # baby澡盆倒入約20ml純露，一般浴缸約50-80ml。
塗抹
  • # 將純露和蘆薈膠混和均勻即可，製成凝膠比例約3:7，精華液比例約8:2。
這樣用最好用
身體保養
  • # 消滅橘皮大作戰：杜松純露＋絲柏純露＋永久花純露外用濕敷，使用海鹽按摩或輕柔按摩皮膚。
肌膚護理
  • # 適合油性或粉刺型肌膚，每日洗完臉後可將杜松純露做為化粧水濕敷使用。
注意事項
  1. 請置於冰箱冷藏保存
  2. 保存期限依瓶身標示，請於期限內使用完畢
  3. 若出現懸浮物質請停止使用
  4. 若需分裝請注意器材務必確實消毒以避免變質
常見問題
Q1：純露可以喝嗎？
A：沐禾有機純露系列皆通過Ecocert有機認證，並未額外添加防腐劑與抗菌劑，建議未經專業芳療師的指示請勿自行服用。
Q2：純露一定要放冰箱保存嗎？
A：純露是蒸餾精油後所得到的有機水溶液，若溫度變化過大或保存不當可能導致細菌滋生或變質，一般建議冷藏保存以維持品質穩定。
