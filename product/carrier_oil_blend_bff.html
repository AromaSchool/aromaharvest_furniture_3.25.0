成分解析
  • # 紅花油：紅花油含有維生素E，木酚素等多種具有抗氧化作用的成分，加上高比例的亞麻油酸，因此運用在肌膚上可以發揮促進血液循環，改善肌膚暗沉、斑點及調理肌膚紋理的作用。
  • # 澳洲胡桃油：澳洲胡桃油主要由棕櫚油酸組成，含豐富的肌膚必需脂肪酸及維生素A、D、E，是植物油中和皮脂接近的油之一，能使肌膚快速得到適度滋潤。
  • # 葵花油：金黃色的葵花油，含有必需脂肪酸以及維生素A、D、E、K，還有鈣、鋅、鉀、鐵和磷等礦物質，具有保濕及軟化肌膚的功效。
  • # 聖約翰草浸泡油(葵花)：聖約翰草浸泡油(葵花)是將聖約翰草的花端放入植物油中浸泡而來，含有豐富金絲桃素、黃酮、丹寧酸、氧大苯甲酮、單烯，對於舒緩肌肉痠痛有不錯的效果。
  • # 雷公根油：別名積雪草，是近幾年在美容界最受歡迎的保養成分，含積雪草苷、三萜烯酸。運用於保養能緊實肌膚線條、淡化細紋，並使肌膚恢復光澤彈性。
  • # 辣木油：使用冷壓工法萃取自有著「奇蹟之樹」美譽的辣木籽，富含高比例的油酸，及維生素A、E，可充分滋潤、溫和調理乾燥與敏弱肌膚。
  • # 榛果油：榛果油含有各種礦物質、維生素A、B1、B2、D、E、卵磷脂和蛋白質，擁有優異的保濕力，也具有激勵循環系統的功效，適合各種類型的肌膚。
使用方式
單用
  • # 取適量沐禾好捧油，塗抹或按摩在需要的部位。使用於臉部約1-3滴，可在掌心勻開之後，再以按壓方式塗抹到臉上。依個人使用習慣增添用量，建議少量多次使用，搭配按摩手式，可幫助油脂吸收、充分滋潤肌膚。
調油
  • # 取分裝瓶(建議PP瓶、玻璃材質較安心)調制精油配方，請按照比例添加想要使用的精油。臉用精華油基底油與精油比例約0.5%~2%，身體按摩油3%、局部按摩油5% 。
乳液
  • # 沐禾好捧油與蘆薈膠以 1:9 的比例調成乳液。作法：沐禾好捧油倒入容器中，再加入精油和蘆薈膠後，以攪拌棒打勻後即可。
油浴
  • # 使用約40ml的沐禾好捧油，均勻塗抹全身後，停留15分鐘再洗淨。也可以在全身塗油之後，進入浴缸泡澡15分鐘。
按摩
  • # 依按摩皮膚面積取適量沐禾好捧油，使用於欲按摩的部位。按摩時的用量比一般潤膚的用量多，臉部約2ml、肩頸約5ml、小腿約3-5ml。均勻塗抹之後再進行按摩動作，可幫助柔軟皮膚、局部加溫、減少皮膚摩擦，增加按摩深度。
卸妝
  • # 使用約2-3ml的沐禾好捧油，均勻覆蓋於肌膚，避開眼周輕柔畫圓按摩，約按摩2分鐘後融化彩妝。用紙巾吸掉多餘油脂後，再使用溫水與洗面乳清潔。(眼妝請使用眼唇卸妝液清潔)
這樣用最好用
臉部保養
  • # 乾性肌膚、受損敏弱肌膚適用，每次使用1-3滴，在掌心勻開之後，再以按壓方式塗抹到臉上。
身體保養
  • # 沐浴後的保養時光，纖腿輕盈排水配方，療癒雙腿一天的辛勞：20ml沐禾好捧油 + 絲柏8滴 + 葡萄柚4滴
緊實輪廓
  • # 再睡10分鐘，揮別早晨泡泡臉的緊實線條配方：10ml沐禾好捧油 + 胡蘿蔔籽2滴
    每次使用1-3滴，在掌心勻開之後，再以按壓方式塗抹到臉上。
  • # 搭配小V臉按摩手法：雙手食指指節微彎，以下巴為中心沿著下顎線往耳際處向上拉提，重複這一個動作，按摩3-5分鐘即可。

常見問題
Q1：基底油可以吃嗎？
A：荷荷芭油屬液態蠟，人體無法代謝，一般基底油與市面上的食用油因萃取與處理方式不同所以不建議口服保健。
Q2：芳療級基底油和超市販售的植物油哪裡不一樣
A：基底油是第一道冷壓萃取的油脂，營養成分完整，而超市裡販售的植物油為了延長保存期限，多數經過特殊處理，應用在芳療上的療效並不大。
